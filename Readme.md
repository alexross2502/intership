#### Установка проекта

---

- Клиентская часть:

```
cd ../web
npm install
npm run
```

- База данных:

```
- Устанавливаем PostgeSQL(v13)
- Создаем пустую БД, устанавливаем значения портов и хоста из ../server/.env.example
```

- Серверная часть:

```
cd ../server
- Создаем файл .env и заполняем недостающие поля данными из Вашей БД
npm install
npm run
```
